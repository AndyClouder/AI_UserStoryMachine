# 题库小程序

一个基于微信小程序的题库应用，支持刷题、收藏和历史记录功能。

## 功能特性

### 题库页面
- 🔍 **搜索功能**: 支持按题目内容和选项搜索题目
- 📝 **答题功能**: 选择答案并查看正确答案和解析
- ⭐ **收藏功能**: 收藏重要题目，方便后续复习
- 📊 **答题记录**: 自动记录答题历史

### 我的页面
- 👤 **用户信息**: 显示用户头像和昵称
- ⭐ **题目收藏**: 查看所有收藏的题目，支持快速跳转
- 📚 **历史记录**: 查看答题历史，支持快速跳转
- 📈 **学习统计**: 显示总题数、收藏数、答题数和完成率
- 🗑️ **数据管理**: 支持清除收藏和历史记录

## 页面布局

### 题库页面布局
- **顶部**: 搜索框
- **中间**: 题目列表（可滚动）
- **底部**: 导航栏

### 我的页面布局
- **顶部**: 用户头像和昵称
- **中间**: 功能列表（收藏、历史记录、统计信息）
- **底部**: 导航栏

## 技术实现

- **框架**: 微信小程序原生开发
- **数据存储**: 本地存储 (wx.setStorageSync/wx.getStorageSync)
- **页面导航**: TabBar 导航
- **样式设计**: Flexbox 布局 + CSS3 动画

## 使用说明

1. **搜索题目**: 在搜索框中输入关键词，可以搜索题目内容和选项
2. **答题**: 点击选项选择答案，点击"查看答案"查看正确答案和解析
3. **收藏题目**: 点击"收藏"按钮收藏当前题目
4. **查看收藏**: 在"我的"页面查看所有收藏的题目
5. **查看历史**: 在"我的"页面查看答题历史记录
6. **跳转题目**: 在收藏和历史记录中点击题目可以快速跳转

## 注意事项

- 需要在微信开发者工具中运行
- 图片资源需要自行添加到 images 目录
- 题库数据可以根据需要在 `utils/data.js` 中修改

## 开发环境要求

- 微信开发者工具
- Node.js 环境（可选，用于构建工具）
